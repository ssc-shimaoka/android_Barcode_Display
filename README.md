# バーコード表示

## 一次元コード概要



## ライブラリ  
 ZXing(ゼブラ・クロッシング)  
 1D/2Dバーコード読み取りライブラリ  
 ※zebra crossingは「横断歩道」の意  

## 対応一次元コードフォーマット  
  1D(製品用)  
  UPC-A  
  UPC-E  
  EAN-8 ‐ JIS X 0501  
  EAN-13 ‐ JIS X 0501  
  1D(産業用)  
  CODE39 ‐ JIS X 0503  
  CODE93  
  CODE128  
  CODABAR(NW-7)  
  ITFコード  
  RSS-14 (全種類)  
  RSS Expanded (殆どの種類)  

## アプリ動作に関して  
  数値を入力後、作成ボタン押下にてバーコードが画面に表示される。  
  クリアボタン押下にて、入力地がクリアされる。
